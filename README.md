# Clinic Management System

A **console-based** clinic appointment management system built in plain **Java**, using Object-Oriented Programming principles. The system simplifies the process of booking appointments by managing patients, doctors, and payments through a terminal interface — no external frameworks or GUI libraries involved.

## Features

- Create and manage patient records
- Create and manage doctor profiles
- Book appointments linking patients to doctors
- Support for multiple payment methods during booking
- Terminal-driven menu system for all operations

## Tech Stack

- **Language:** Java (core, no external GUI framework)
- **Paradigm:** Object-Oriented Programming

## Design

The system is modeled around core entities — patients, doctors, and appointments — with classes responsible for their own data and behavior, following OOP principles like encapsulation and separation of concerns. All interaction happens through the console, keeping the project focused on backend logic rather than UI.

## Getting Started

### Prerequisites
- JDK installed (Java 8+)

### Run

```bash
javac Main.java
java Main
```

*(Adjust the entry-point filename above to match the actual main class in the repo.)*
